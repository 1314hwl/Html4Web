列表

  1. 无序列表 - 列表项标记用特殊图形（如小黑点、小方框等）
  2. 有序列表 - 列表项的标记有数字或字母

不同的列表项标记:list-style-type

  • list-style-type: circle;
  • list-style-type: square;
  • list-style-type: upper-roman;
  • list-style-type: lower-alpha;

作为列表项标记的图像:list-style-image

  • list-style-image: 图标;
  • list-style-image: 图标;
  • list-style-image: 图标;
  • list-style-image: 图标;
